# getting help

- **bug or feature request**: [open an issue](https://github.com/sunnypatell/sunnify-spotify-downloader/issues/new/choose) - the forms route you
- **question**: use the [question template](https://github.com/sunnypatell/sunnify-spotify-downloader/issues/new/choose); check [README troubleshooting](README.md#troubleshooting) first
- **security issue**: do NOT open a public issue - see [SECURITY.md](SECURITY.md)
- **diagnostics**: `python scripts/check_api_status.py` tells you whether the embed API or yt-dlp is the problem before you file

solo-maintained; issues are auto-assigned and triaged in batches. include OS, app version, and the exact input that failed - reports with artifacts get fixed dramatically faster.
